# AdaptIQ — Adaptive Learning Recommendation System

A research-backed adaptive learning recommendation system that analyzes student profiles to generate personalized learning pathways.

## Quick Start — Preview the Web Page

### Option 1: Open Directly in Browser (Simplest)
1. Open **File Explorer** and navigate to this folder
2. Double-click `index.html`
3. The page will open in your default browser

### Option 2: Right-Click Menu
1. Right-click `index.html` in VS Code
2. Select **"Open in Default Browser"**

### Option 3: Use Live Server Extension (Recommended for Development)
1. Install the **Live Server** extension in VS Code
   - Go to Extensions (Ctrl+Shift+X)
   - Search for "Live Server" (by Ritwick Dey)
   - Click Install
2. Right-click `index.html` and select **"Open with Live Server"**
3. The page opens at `http://localhost:5500` with auto-refresh on file changes

### Option 4: Manual Browser URL
1. Copy the full path to `index.html` from your file explorer
2. Paste into your browser address bar (it should start with `file:///`)
3. Press Enter

## Files

- `index.html` — Main page structure
- `style.css` — Styling and animations
- `app.js` — Quiz logic, scoring, and recommendations
- `README.md` — This file
